# 📊 Data Visualization Project

## Overview

This project demonstrates how to analyze and visualize population data using *Python, **Pandas, and **Matplotlib*. It covers data exploration, pivot tables, visualization techniques, and exporting results.

## Files

- population_total.csv – Population dataset
- 1.Dataset Overview and Making Pivot Table.ipynb
- 2.Data Visualization with Pandas.ipynb
- 3.Save Plot and Export Pivot Table.ipynb
- Interactive Visualization with Pandas.ipynb

## Topics Covered

- Data exploration with Pandas
- Creating pivot tables
- Line and bar charts
- Interactive visualizations
- Saving plots and exporting data
